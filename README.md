## 🚀 Getting Started

Clone the repository:

```bash
git clone https://github.com/Bayaankhan/C-Plus-Plus-Language.git
```

Navigate to the repository:

```bash
cd C-Plus-Plus-Language
```

Compile a program:

```bash
g++ filename.cpp -o output
```

Run the program:

```bash
./output
```
